# ToxPredict-hERG — MLflow Tracking Server

MLflow tracking server for the **ToxPredict-hERG** project, a machine learning pipeline predicting hERG channel inhibitory activity (pIC50) from molecular structures.

## What this Space does

Hosts a persistent MLflow UI to track and compare experiments across feature sets (Lipinski descriptors, Morgan fingerprints, MACCS keys) and models (Ridge, Random Forest, XGBoost, MLP, GNN).

## Project

The full project is available on [GitHub](https://github.com/Adrien-COSSON/ToxPredict-hERG).
